# Assets

Drop the game art here as PNGs with transparent backgrounds. Suggested names so the code can find them:

- `hero.png` — the samurai gatekeeper
- `oni-1.png`, `oni-2.png` — disguised oni variants
- `villager-1.png`, `villager-2.png`, `villager-3.png` — honest villagers
- `guard.png` — the defense token placed on grid tiles
- `background.png` — night scene (or keep this as CSS/SVG and skip the file)

Export at roughly twice the on-screen display size so they stay sharp on high-density phone screens. Prompt starters for generating these are in `../DESIGN.md`.
